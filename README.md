## Description

ZeroNest is a boilerplate that has most of the common web-dev features built-in.

## Installation

Clone the repo and run

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn dev

# enable debugger
$ yarn start:debug

# production mode
$ yarn start:prod
```
